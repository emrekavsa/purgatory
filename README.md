# Purgatory

Purgatory is a full-stack social polling app built with Next.js and Supabase. It lets users create text or image polls, vote once per poll, discuss in threaded comments, report content, and moderate reports through an admin dashboard.

## What Was Built

- Public poll feed with categories, sorting, and search
- Text and image-based poll creation
- Threaded comments with edit and delete flows
- User profiles with avatar uploads
- Content reporting for polls and comments
- Admin dashboard for report dismissal and user bans
- RLS-backed permissions for user and admin actions

## Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Supabase Auth, Database, and Storage
- Tailwind CSS 4
- Valibot
- Vercel
